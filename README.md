> show your love on this valentine's day through console.

## Install

```
$ npm install --global i-love
```

## Usage

```
$ i-love

  Usage :

    $ i-love -u UNICORN -c rainbow
    
    Options:

  -u  Name                                                            [required]
  -c  Set Colors                                                      [required]


```

## Extra

> Extra colors available ( execpt rainbow ) :

```
- red									- magenta
- green									- gray
- yellow								- grey
- blue									- zebra
- cyan									- america
- white									- random
- black 							        - trap

```
